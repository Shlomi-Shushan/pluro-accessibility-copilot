# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Pluro Accessibility Copilot, please report it responsibly:

- **Do not disclose publicly.**
- Email the security team at: security@pluro.ai
- Provide as much detail as possible (steps to reproduce, environment, impact).
- We will acknowledge receipt within 2 business days and provide a timeline for remediation.

## Supported Versions
We support the latest major version of the extension. Please update to the latest version for security fixes.

## Disclosure Process
- We will investigate and fix valid vulnerabilities as quickly as possible.
- Once resolved, we will credit the reporter (if desired) and publish a security update.
